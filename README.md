# satellite_presentation
 A web quiz that determines "What kind of Satellite are you?" based on your personality traits. Built with vanilla HTML, CSS, and JavaScript.
 A little database of satellites.
 A not technical simulation of the Viasat attack.
 A web quiz testing basic knowledge of satellites.
 A "Build your own satellite" game. Outputs of built satellites need to be thoroughly checked.

 The base structure was made by Perplexitity Ai, then I adjusted, reassembled and added some parts to it.
